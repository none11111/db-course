## 项目说明

### 论坛数据分析平台 (dbcourse-after/)

这是一个基于多维度优化的论坛数据分析平台，旨在为论坛运营者提供全面的数据洞察和决策支持。

#### 核心功能
- **数据仪表盘**：提供概览统计、趋势分析、分类分布和用户分析等可视化图表
- **数据管理**：用户信息管理、内容管理和数据验证
- **性能优化**：索引优化、存储过程和高效SQL查询实现

#### 技术栈
- **后端**：Flask + SQLAlchemy + MySQL
- **前端**：HTML5 + CSS3 + JavaScript + Bootstrap 5 + ECharts
- **开发工具**：Git + MySQL Workbench

## 使用方法

1. 进入项目目录：`cd dbcourse-after`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行应用：`python run.py`
4. 访问应用：`http://localhost:5000`
